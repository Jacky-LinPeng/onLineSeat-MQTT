# 在线选座

*******************
环信MQTT参赛作品

## 运行环境
iOS 9.0+

## 功能简介
多台iOS设备进入app,即可同步选位(基于环信MQTT).

## 资源说明
图标，ICON，封面等资源均为开发使用，并非商业用途。
